# Title: Exploratory-Data-Visualization-using-Matplotlib-Seaborn-Plotly

##  Objective

The objective of this project is to perform exploratory data analysis (EDA) on a synthetic retail dataset using data visualization techniques. The goal is to select appropriate charts, present them clearly, and derive meaningful insights.

## Problem Statement

The task is to explore a retail dataset and answer analytical questions using different types of visualizations. Each visualization is created using the most suitable library and includes a brief observation to simulate a real-world EDA workflow.

## 🛠 Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly

## 📊 Dataset Description

The dataset is synthetically generated and contains the following features:

* **Age** → Customer age
* **Income** → Annual income of the customer
* **Spend** → Amount spent by the customer
* **Category** → Product category (Electronics, Clothing, Groceries, Sports)

## Tasks Performed

### Task 1: Bar Chart — Average Spend by Category

* Calculated average spending for each category
* Visualized using Matplotlib bar chart
* Identified the category with the highest average spend

###  Task 2: Scatter Plot — Income vs Spend

* Created a scatter plot using Seaborn
* Analyzed the relationship between income and spending behavior
  
### Task 3: Interactive Scatter Plot — Plotly

* Built an interactive scatter plot using Plotly
* Colored data points by category
* Explored insights using hover and zoom features

## Key Insights
* Certain product categories show higher average spending than others
* The relationship between income and spending is not strongly linear
* Interactive visualization provides deeper insights through user interaction
  
Conclusion

This project demonstrates how to effectively use different visualization tools to analyze data and extract insights. It highlights the importance of choosing the right chart type and presenting findings clearly.
